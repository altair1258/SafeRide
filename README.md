# SafeRide Guardian

Smart Vehicle Safety System with real-time GPS tracking, accident detection, and emergency alerts.

## Features

- Multi-vehicle support (Car and Scooter modes)
- Real-time 3D vehicle orientation visualization
- GPS location tracking with 24-hour travel history
- Vehicle-specific accident detection algorithms using MPU9250 sensors
- Dynamic connection status monitoring
- Emergency alert system via Telegram
- Live sensor data streaming from ESP32
